# ABC Body Style Guide

This is model guidance, not a deterministic backend template.

Write one continuous paragraph for each billable hour. A strong paragraph normally includes:

- a concrete activity, setting, material, or instruction;
- an observable client action and a behavior name drawn from the profile/assessment;
- one or more interventions drawn from the profile/assessment, with how the RBT applied them;
- an observable response;
- the exact replacement-program name assigned in the app for that hour;
- how the RBT taught or prompted that program;
- the exact percentage selected in the app.

The model may vary sentence structure and may use multiple interventions. The backend does not
function-match, reorder, remap, or rewrite clinical content.
