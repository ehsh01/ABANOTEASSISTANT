# Sample ABC Style Reference

The four user-supplied notes from July 20, 2026 are the style reference for model-written ABC
paragraphs. They demonstrate:

- concrete materials and instructions;
- observable behavior descriptions;
- natural use of one or multiple interventions;
- observable responses;
- exact replacement-program names;
- exact manually selected percentages.

They are examples, not server templates. The application continues to use the current locked opening
(no location) and current locked closing (plain `praise`), even where a supplied sample used older
wording.
